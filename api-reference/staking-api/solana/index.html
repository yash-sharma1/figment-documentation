Authenticationâ€‹

API calls require authentication with API key via Authorization request header or URL property. Calls are made to the Solana Staking API endpoint, hosted at https://solana-slate.datahub.figment.io

Here's some boilerplate to get you started.

fetch("https://solana-slate.datahub.figment.io", {
    method: 'POST', // can also be 'GET', 'PUT' or any appropriate method
    headers: {
        "Authorization" : "<api_key>",
        "Content-Type": "application/json" // if method is 'POST'
    },
    body: JSON.stringify({
      // JSON Payload
    })
})
