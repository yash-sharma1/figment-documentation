Staking API - NEAR

Authentication​

API calls require authentication with API key via Authorization request header or URL property. Calls are made to the Near Staking API endpoint, hosted at https://near-slate.datahub.figment.io

Here's some boilerplate to get you started.
  • Fetch with Header
  • Fetch with Parameter
  • CURL with Header
  • CURL with Parameter
fetch("https://near-slate.datahub.figment.io", {
    method: 'POST', // can also be 'GET', 'PUT' or any appropriate method
    headers: {
        "Authorization" : "<api_key>",
        "Content-Type": "application/json" // if method is 'POST'
    },
    body: JSON.stringify({
      // JSON Payload
    })
})
fetch("https://near-slate.datahub.figment.io", {
    method: 'POST', // can also be 'GET', 'PUT' or any appropriate method
    headers: {
        "Content-Type": "application/json" // if method is 'POST'
    },
    body: JSON.stringify({
      // JSON Payload
    })
})
curl -X POST 'https://near-slate.datahub.figment.io' \
--header 'Content-Type: application/json' \
--header 'Authorization: <api_key>' \
--data '{
  // JSON Payload
}'
curl -X POST 'https://near-slate.datahub.figment.io' \
--header 'Content-Type: application/json' \
--data '{
  // JSON Payload
}'
