Staking API - Polkadot

Authentication​

API calls require authentication with API key via Authorization request header or URL property. Calls are made to the Polkadot Staking API endpoint, hosted at https://polkadot-slate.datahub.figment.io

Here's some boilerplate to get you started.
  • Fetch with Header
  • Fetch with Parameter
  • CURL with Header
  • CURL with Parameter
fetch("https://polkadot-slate.datahub.figment.io", {
    method: 'POST', // can also be 'GET', 'PUT' or any appropriate method
    headers: {
        "Authorization" : "<api_key>",
        "Content-Type": "application/json" // if method is 'POST'
    },
    body: JSON.stringify({
      // JSON Payload
    })
})
fetch("https://polkadot-slate.datahub.figment.io", {
    method: 'POST', // can also be 'GET', 'PUT' or any appropriate method

  // JSON Payload
}'
