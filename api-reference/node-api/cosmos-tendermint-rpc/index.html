Node API - Cosmos Tendermint RPC


API AuthenticationClick to view API Authentication details
API calls require authentication with API key via Authorization request header or URL property. Calls are made to the Cosmos Tendermint Rpc Node API endpoint, hosted at https://cosmoshub-4--rpc--archive.datahub.figment.io

Here's some boilerplate to get you started.
  • Fetch with Header
  • Fetch with Parameter
  • CURL with Header
  • CURL with Parameter
fetch("https://cosmoshub-4--rpc--archive.datahub.figment.io/health", {
    method: 'POST', // can also be 'GET', 'PUT' or any appropriate method
    headers: {
        "Authorization" : "<api_key>",
        "Content-Type": "application/json" // if method is 'POST'
    },
    body: JSON.stringify({
      // JSON Payload
    })
})
fetch("https://cosmoshub-4--rpc--archive.datahub.figment.io/apikey/{{apikey}}/health", {
    method: 'POST', // can also be 'GET', 'PUT' or any appropriate method
    headers: {
        "Content-Type": "application/json" // if method is 'POST'
    },
    body: JSON.stringify({
      // JSON Payload
    })
})
curl -X POST 'https://cosmoshub-4--rpc--archive.datahub.figment.io/health' \
--header 'Content-Type: application/json' \
--header 'Authorization: <api_key>' \
--data '{
  // JSON Payload
}'
curl -X POST 'https://cosmoshub-4--rpc--archive.datahub.figment.io/apikey/{{apikey}}/health' \
--header 'Content-Type: application/json' \
--data '{
  // JSON Payload
}'
