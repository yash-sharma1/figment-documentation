    headers: {
        "Authorization" : "<api_key>",
        "Content-Type": "application/json" // if method is 'POST'
    },
    body: JSON.stringify({
      // JSON Payload
    })
})

dag_getHeads

dag_getHeads returns IDs of all the epoch events with no descendants in a given epoch.

Request Body (try it)
{
  "jsonrpc": "2.0",
  "method": "dag_getHeads",
  "params": [
    "pending"
  ],
  "id": 1
}
Example Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": [
    "0x0002078c00000e08a7a882aa37937ccb2253c7d29a588715ad2b9d0c5530e3c4"
  ]
}
