To refresh the transaction state and get the latest status of the existing flow we'll be using the flow ID from the previous step.

### Parameters

- None

#### Sample cURL Request

```bash
curl -X GET 'https://near-slate.datahub.figment.io/api/v1/flows/5f0962d5-fba0-453c-bf35-0dad79624b59' \
-H 'Authorization: <api_key>'
```

### Response

- `state` &mdash; `string` The current state of the flow.

:::info note

The `state` field will update from `transfer_tx_broadcasting` to `transfered`.

:::
