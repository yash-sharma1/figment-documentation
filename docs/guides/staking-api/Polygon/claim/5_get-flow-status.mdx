To get the current state of the existing flow, send a `GET` request to `/api/v1/flows/[:flow_id]` using the flow ID from the previous step.

### Parameters

- None

#### Sample cURL Request

```bash
curl -X GET 'https://polkadot-slate.datahub.figment.io/api/v1/flows/86d9a004-12b6-4142-90cb-9ade53864e29' \
-H 'Authorization: <api_key>'
```

### Response

- `state` &mdash; `string` The current state of the flow.

:::info note

The `state` field will update from `claim_rewards_tx_broadcasting` to `done`.

:::
