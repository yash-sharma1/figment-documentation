To get the current state of the existing flow, send a `GET` request to `/api/v1/flows/[:flow_id]` using the flow ID from the previous step.

### Parameters

- None

#### Sample cURL Request

```bash
curl -X GET 'https://polkadot-slate.datahub.figment.io/api/v1/flows/07d6752c-c412-4cfa-a12c-65feab06acd6' \
-H 'Authorization: <api_key>'
```

### Response

- `state` &mdash; `string` The current state of the flow.

:::info note

The `state` field will update from `claim_tx_broadcasting` to `done`.

:::
